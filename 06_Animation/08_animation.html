animation

키프레임 규칙을 사용해서 움직이는 동작을 만듦

animation:name duration timing-function delay count direction;

*name 키프레임 규칙의 이름

@keyframes name{ }

가로 600/세로 400/핑크~보라..3초

count 반복횟수

한 번 재생(기본값) / 숫자 1, 2.. / 무한반복 infinite

direction 진행방향

normal(기본값) - 처음(0%) → 끝(100%)

alternate - 처음(0%) → 끝(100%) → 처음(0%)
